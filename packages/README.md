# Gemeinsame Pakete

Gemeinsame Pakete enthalten versionierte Verträge, Datentypen und reine, gemeinsam getestete Regelpakete.

- `contracts/` enthält Befehls- und Ereignishüllen für die spätere Client-Server-Kommunikation.

Gemeinsame Pakete heben die Trennung der ausführbaren Client- und Serveranwendungen nicht auf.
