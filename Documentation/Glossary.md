# Glossar

## Zweck

Das Glossar sorgt dafür, dass Regeln, Oberfläche und spätere Implementierung dieselben Begriffe verwenden.

| Begriff | Bedeutung | Status |
|---|---|---|
| Escalera | Name des Spiels und des Projekts | Bestätigt |
| Partie | Ein vollständiges Spiel vom Start bis zum Ende | Entwurf |
| Runde | Zusammengehöriger Abschnitt, in dem mehrere Spieler zum Zug kommen können | Offen |
| Zug | Spielabschnitt eines einzelnen aktiven Spielers | Entwurf |
| Aktiver Spieler | Spieler, der aktuell handeln darf | Entwurf |
| Spielstand | Gesamtheit der momentan gültigen Spieldaten | Entwurf |
| Aktion | Einzelne erlaubte Handlung eines Spielers | Entwurf |
| Spielvariante | Bewusste Abweichung von den Standardregeln | Entwurf |
| Lobby | Bereich, in dem 2 bis 6 Spieler zusammenkommen und die Partie konfigurieren | Bestätigt |
| Nachziehstapel | Verdeckter Stapel, von dem der aktive Spieler eine Karte ziehen kann | Bestätigt |
| Ablagestapel | Offener Stapel in der Mitte; seine oberste Karte kann gezogen oder gekauft werden | Bestätigt |
| Kombination | Gültige Gruppe gleicher Kartenwerte oder gültige Straße | Bestätigt |
| Straße | Folge aufeinanderfolgender Kartenwerte | Bestätigt |
| Escalera | Straße aus sieben Karten derselben Farbe in Phase 7 | Bestätigt |
| Kartenkauf | Erwerb der aktuell abgelegten Karte durch einen nicht aktiven Spieler gegen eine Münze | Teilweise offen |
| Farbe / Zeichen | Kreuz, Pik, Herz oder Karo | Bestätigt |
| Bereit | Lobby-Status, mit dem ein Spieler dem Start der Partie zustimmt | Bestätigt |
| Gemeinsame Phase | Aktuelle Pflichtphase, die gleichzeitig für alle Spieler gilt | Bestätigt |
| Phasenende | Erfolgreicher Abschluss der gemeinsamen Phase durch mindestens einen Spieler | Bestätigt |
| Wiedereintritt | Rückkehr eines getrennten Spielers in seine laufende Lobby und Partie | Bestätigt |
| Strafpunkte | Summe der Werte der bei Rundenenden verbliebenen Handkarten | Bestätigt |
| Stärkere Kombination | Phasenkombination mit mehr passenden Karten als mindestens gefordert | Bestätigt |
| Toter Stapel | Alternative Bezeichnung für den offenen Ablagestapel | Bestätigt |

Weitere Begriffe werden ergänzt, sobald bei Planung oder Umsetzung neue Fachbegriffe entstehen.
