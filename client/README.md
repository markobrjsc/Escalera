# Escalera Client

Der Client ist eine eigenständige React-/Vite-Anwendung. Er wird getrennt gebaut und in einem eigenen Nginx-Dockercontainer ausgeliefert.

Er besitzt ein Web-App-Manifest, einen Service Worker und eine gerätespezifische Installationshilfe. Installiert startet Escalera im Vollbild- beziehungsweise Standalone-Modus ohne Browser-Navigationsleisten, soweit das Betriebssystem dies erlaubt.
